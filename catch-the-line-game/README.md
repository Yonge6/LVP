# 接住这句话｜虾子曰英语冒险小游戏

这是一个无依赖、手机优先的网页英语小游戏，围绕“听懂对方意图，并立即用英语把话接下去”设计。

视觉风格已按“欢快、轻松、时尚、适合 00 后审美”重做，并接入虾子曰与豆豆龙本地品牌素材。

## 已实现

- 虾子曰、豆豆龙真实角色素材展示
- 双角色专属技能：虾式救场、回声龙息
- 地图、关卡、5 回合战斗与 Boss
- 浏览器英文朗读
- 浏览器语音识别（支持时）
- 无语音识别时的快捷回应模式
- 意图判断、倒计时、生命值、连击、伤害、粒子奖励
- 正反馈结算页
- 本地保存宝石与连续天数
- 手机安全区与响应式布局

## 本地体验

方式一：直接打开 `index.html`。

方式二：在当前文件夹启动本地服务后访问：

`http://127.0.0.1:4173/`

语音识别通常要求 HTTPS 或 localhost；部署到 GitHub Pages 后使用体验更完整。

## 推荐部署路径

把 `catch-the-line-game` 整个文件夹上传到仓库，例如：

`OneLaser/catch-the-line/index.html`

部署后地址可类似：

`https://yonge6.github.io/OneLaser/catch-the-line/`

## 后续产品化建议

1. 把角色素材升级为透明 PNG / Spine / Lottie 动画，战斗表现会更顺滑。
2. 接入大模型语义判断，不再依赖关键词。
3. 接入语音识别与发音反馈服务，支持重音、连读、节奏建议。
4. 增加分支剧情、装备卡、每日副本、好友助战和赛季任务。
5. 关卡内容由后台 JSON 配置，方便每天生成新剧情。
