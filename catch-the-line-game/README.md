# 虾子曰：回声之境｜Echoes of the Speaking Valley

手机优先的 H5 第一关可玩原型，关卡名为《沉默之门 / The Silent Gate》。

## 已实现

- 竖屏等距悬浮建筑场景
- 虾子曰角色入场与移动
- 点击落脚点移动
- 点击中央转台旋转建筑并连接道路
- 石碑播放 “Can you hear me?”
- 门前语音交互，说出或模拟 “Hello” 开门
- 世界苏醒、莲花开放、回声碎片结算页
- 无语音识别时的“替我说 Hello”兜底

## 本地体验

直接打开 `index.html`，或在目录内启动本地服务后访问：

`http://127.0.0.1:4173/`

语音识别通常要求 HTTPS 或 localhost。
